📖 Application Help & User Guide

Welcome to your L2 Operations Command Center. This tool is designed to track tasks, incidents, and renewals entirely within your browser without requiring a backend server.

🔗 Data & Connectivity

  • The JSON Database: All data (tasks, settings, logos) is stored in a single .json file.
  • Linking the File: Click Link JSON at the top right. Requires Chrome or Edge desktop browsers. Once linked, the app reads the file and will automatically overwrite it silently every 30 seconds and upon any manual edits.
  • Backups: Go to Settings > Data Management to export a fresh JSON backup to your downloads folder at any time.

✅ Task Management & Workflows

  • SLA Automation: In Settings, assign SLA days to your priorities. When you add a task, the ETA will automatically calculate based on the open date.
  • Smart Reminders: Changing a task type to "Renewal" will automatically calculate 30, 45, and 60-day reminders based on the ETA.
  • Pinning (📌): Click the pin icon next to any active task ID to lock it permanently to the top of the grid.
  • Archiving/Closing: When you change a task's status to Closed, you are required to enter Closure Notes. The task will migrate out of the active manager into the History ledger.

📊 Dashboards & Visualization

  • Interactive Charts: Clicking on legend items in any chart toggles that specific data point on and off to filter your view.
  • Advanced Customizer: In Settings, use the Advanced Chart Customizer to target specific charts. You can change a Doughnut chart to a Bar chart, toggle grid lines, or apply a custom background to highlight critical metrics.
  • Gantt Hover Context: Hovering over any task bar on the Gantt chart provides a custom popup detailing the description, assignee, and exact timeline dates.

📥 Bulk Excel Import

  • Formatting: The easiest way to import bulk updates is to Export Workbook, edit the rows in Excel, and re-import that exact file.
  • Multi-Sheet: The export generates 3 sheets: Tasks, Team, and Dictionaries. You can edit configurations and colleagues directly in Excel before re-importing.
  • Intelligent Merging: If an imported Task ID matches an existing record, the app updates it. Assignee names are automatically matched to internal Team profiles.

🔔 Smart Notifications

  • Red Alerts (🔴): Any active task that is due today or overdue based on its ETA.
  • Amber Alerts (🟠): "Renewal" type tasks that are due within the next 30 days.
  • Yellow Alerts (🟡): "Renewal" type tasks that are due within the next 31 to 60 days.
  • Click the Bell icon in the header to view a detailed modal list of all triggered alerts.
